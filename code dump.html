How it works.

Super easy to use

Steps

#1

An online order is placed.

#2

On the checkout page, THE DROP. option is selected [targeting e- commerce] .

#3

Based on the order ID a One Time Password is generated.

#4

The One Time Password is used by the delivery executive to deliver the package.

#5

The user can collect the parcel inside the box at any point via the app[single delivery at a time].
